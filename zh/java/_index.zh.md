GroupDocs.Comparison 使您能够区分 Microsoft Word、Excel、PowerPoint、openDocument ODT、PDF、文本和 HTML 文档。

文档比较 API 比较内容以检测单词、段落和字符的变化。

它还比较字体大小、粗体、斜体等样式更改。差异摘要保存在单独的结果文件中。
