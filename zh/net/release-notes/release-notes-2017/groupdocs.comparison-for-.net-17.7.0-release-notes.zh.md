## 主要特点

在这个每月定期发布的版本中有 4 个新功能、2 个改进和 3 个修复。最值得注意的是：

* 引入了对所有格式的样式设置的支持
* 引入所有格式的克隆密码设置
* 为所有格式的克隆元数据引入设置
* 增加了细胞格式的克隆能力
