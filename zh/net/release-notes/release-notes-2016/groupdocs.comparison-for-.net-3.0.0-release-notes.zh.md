## 主要特点

这是 .NET 的新一代 GroupDocs.Comparison 的第一个版本。最显着的特点是：

* 所有格式的比较引擎已从头开始完全重写。
* 简化且非常灵活的 API。
* 更好的组织和简化的产品结构。
* 代码优化。
*应用/放弃所有格式的更改支持
* 设计为本地 .NET 库，允许在所有可能的项目类型中使用。
