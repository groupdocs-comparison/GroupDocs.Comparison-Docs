## 主要特点

在这个每月定期发布的版本中有 12 项改进和修复。最值得注意的是：

* 改进了比较表的比较.Words
* 改进了Comparison.PDF的表格比较
* 改进了 Compare.Cells 的性能
* 改进了 Compare.Words 内容列表的比较
