## 概述

**[GroupDocs.Comparison for .NET](https://products.groupdocs.com/comparison/net)** 不需要安装任何外部软件或第三方工具。只需按照 [安装]({{< ref "comparison/net/getting-started/installation.md" >}}) 中描述的一种方式。

## 支持的操作系统

GroupDocs.Comparison for .NET 支持安装了 .NET 或 Mono 框架的任何 32 位或 64 位操作系统，包括但不限于：

### 窗户

* Microsoft Windows 2003 服务器（x64、x86）
* Microsoft Windows 2008 服务器（x64、x86）
* Microsoft Windows 2012 服务器（x64、x86）
* Microsoft Windows 2012 R2 服务器（x64、x86）
* Microsoft Windows 2016 服务器（x64、x86）
* Microsoft Windows 2019 服务器（x64、x86）
* Microsoft Windows Vista（x64、x86）
* Microsoft Windows XP（x64、x86）
* 微软视窗 7 ( x64, x86)
* Microsoft Windows 8、8.1（x64、x86）
* 微软视窗 10 (x64, x86)
*微软天青

### Linux

* Linux（Ubuntu、OpenSUSE、CentOS 等）

＃＃＃ 苹果电脑

* Mac OS X
