# GroupDocs.Comparison - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Comparison](https://docs.groupdocs.com/comparison/) static documentation.

## About GroupDocs.Comparison

[GroupDocs.Comparison](https://products.groupdocs.com/comparison/) is a document comparison SDK that enables you to detect differences between two file versions across multiple formats including PDF, Word, Excel, and PowerPoint.

Available for .NET, Java, Node.js, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/comparison/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
