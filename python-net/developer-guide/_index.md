[GroupDocs.Comparison for Python via .NET](https://products.groupdocs.com/comparison/python-net) exposes a small, focused API surface organised across three groups of tasks:

- **[Comparing documents]({{< ref "comparison/python-net/developer-guide/comparing-documents" >}})** — compare two or more documents, tune sensitivity, customize change styles, accept or reject detected changes, and handle format-specific scenarios (Word, PDF, JSON, Markdown, and more).
- **[Loading documents]({{< ref "comparison/python-net/developer-guide/loading-documents" >}})** — feed inputs into `Comparer` from file paths, streams, raw strings, password-protected files, or directories with custom fonts.
- **[Saving comparison results]({{< ref "comparison/python-net/developer-guide/saving-results" >}})** — pick the output format, choose where metadata comes from, and apply a password to the result document.

Plus four cross-cutting topics:

- [Get document information]({{< ref "comparison/python-net/developer-guide/getting-document-info" >}}) — inspect file type, size, and page count without performing a full comparison.
- [Get supported file formats]({{< ref "comparison/python-net/developer-guide/get-supported-file-formats" >}}) — enumerate the supported format list at runtime.
- [Compare folders]({{< ref "comparison/python-net/developer-guide/compare-folders" >}}) — diff entire directories instead of single documents.
- [Generate document pages preview]({{< ref "comparison/python-net/developer-guide/generate-document-pages-preview" >}}) — render thumbnails of selected pages.
- [Logging and diagnostics]({{< ref "comparison/python-net/developer-guide/logging-and-diagnostics" >}}) — capture progress messages, warnings, and errors for debugging or audit.
