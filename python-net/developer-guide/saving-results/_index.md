[GroupDocs.Comparison](https://products.groupdocs.com/comparison/python-net) supports several ways to control how the result document is saved. The [`SaveOptions`](https://reference.groupdocs.com/comparison/python-net/groupdocs.comparison.options.save/SaveOptions) class exposes these properties:

- **`clone_metadata_type`** — choose whether to clone metadata from the source, target, or neither.
- **`file_author_metadata`** — set author/title/created-on metadata on the result document.
- **`password`** — apply a password to the result document for protected output.

For details, see the guides linked below.
